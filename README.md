# E-Commerce Analytics Dashboard

## Overview

An interactive Power BI dashboard built to analyze E-Commerce business performance.

## Features

* Executive Overview Dashboard
* Sales Analysis
* Customer Insights
* Product Performance Analysis
* Profit & Forecast Analysis

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Key KPIs

* Total Sales
* Total Profit
* Product Category Analysis
* Monthly Profit Trends
* Forecasting & Goal Tracking

1. Overview
2. Sales Analysis
3. Customer Insights
4. Product Analysis
